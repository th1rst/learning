My very own Tables-Cheatsheet

Syntax Description
<table> </table> Starts the table. All data goes in here.
<thead> </thead> Heading of the table. Not necessary, but can be styled seperately with CSS
<tbody> </tbody> Table Body. Not absolutely necessary (gets rendered in the DOM anyway) but still useful.
<tr> </tr> Table Row. Must be inserted for every new row. No Data input.
<td> </td> The actual table content/data goes in here.
<tfoot> </tfoot> Table Footer. Represents the bottom of the table.
